# Customer Retention & Revenue Analysis

## Project Overview

This project analyzes an online retail dataset to uncover insights into customer behavior, revenue trends, and retention patterns.

The goal is to simulate a real-world business scenario by combining **Python, SQL, and Power BI** in a single workflow.

---

## Tools Used

* **Python** – Data cleaning and preprocessing
* **SQL (MySQL)** – Data analysis and feature engineering
* **Power BI** – Data visualization and dashboard creation

---

## Dataset

* UK Online Retail Dataset
---

## Project Workflow

### 1.Data Cleaning (Python)

* Removed missing values
* Created new columns:

  * `TotalAmount`
  * `YearMonth`
  * `DayOfWeek`
* Prepared dataset for SQL analysis

---

### 2.Data Analysis (SQL)

Performed advanced analysis including:

* Revenue trends

* Customer spending behavior

* RFM segmentation:

  * Recency
  * Frequency
  * Monetary

* Cohort analysis for customer retention

---

### 3.Dashboard (Power BI)

The dashboard includes:

* 📈 Monthly Revenue Trend
* 🌍 Revenue by Country
* 👥 Customer Segmentation (RFM)
* 🧾 Top Customers
* 🔥 Cohort Retention Heatmap

## Key Insights

* Majority of revenue comes from a small segment of high-value customers
* Customer retention declines over time (visible in cohort analysis)
* Certain countries dominate overall revenue contribution
